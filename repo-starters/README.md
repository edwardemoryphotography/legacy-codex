# Repo Starter Packs

This directory contains generated starter repositories for consolidation targets.

## Generated Targets

- `codex-core`: Execution governance + architecture source of truth
- `neurofeedback-stack`: Real-time EEG and biometrics integration platform
- `memory-intelligence`: Persistent memory and learning extraction platform
- `agent-tooling-hub`: Agent interfaces and artifact capture utilities

## Regenerate

```bash
python3 scripts/generate_repo_standards.py --output-dir repo-starters --overwrite
```
