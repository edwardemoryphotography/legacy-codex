# memory-intelligence

> Persistent memory and learning extraction platform

## Overview

Combine scoped memory persistence, retrieval APIs, and cross-session preference extraction.

## Project Status

- **Phase**: Standardization bootstrap
- **Primary stack**: Python services + CLI/Desktop clients
- **First milestone**: Store and retrieve session memories with project/user isolation and documented retention policy.
