# Security Policy

## Project Scope

`muse-neurofeedback`: Neurofeedback app for Muse EEG with real-time visualization.

## Reporting a Vulnerability

Please report vulnerabilities privately:
- Use GitHub private security advisories if enabled, or
- Contact maintainers directly: @edwardemoryphotography

Include:
1. Affected area and potential impact
2. Reproduction steps
3. Suggested mitigation (if known)

## Response Targets

- Initial acknowledgment within 72 hours
- Mitigation plan within 7 days (target)
